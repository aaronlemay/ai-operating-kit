# Examples

Worked, end-to-end walkthroughs showing how to combine AI Operating Kit skills on realistic studio and operator problems. Each example names which skills to run, in what order, with short illustrative excerpts of the inputs and outputs.

**AI is not the job. AI is the leverage layer.** These examples show the leverage layer at work — scoping, checking, and moving a decision forward — while a qualified human stays accountable for the call.

> **All examples are fictional and for illustration only.** Company names, numbers, people, and transactions are invented and generalized. Nothing here is legal, tax, financial, or accounting advice. Anything involving money, contracts, equity, or people requires qualified professionals.

## How to read an example

Each file follows the same shape:

1. **Scenario** — the fictional situation and who needs a decision.
2. **Skill sequence** — the ordered list of skills to run and why.
3. **Walkthrough** — short excerpts of inputs and outputs at each step.
4. **Guardrails** — what the workflow does *not* do, and where a human or professional must own the call.

Run the skills in the order shown. Adapt the numbers and names to your own generalized, non-confidential inputs.
